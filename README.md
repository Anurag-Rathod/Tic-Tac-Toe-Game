# 🎮 Tic Tac Toe Game

A responsive Tic Tac Toe game built with HTML, CSS, and JavaScript featuring an interactive UI, turn-based gameplay, automatic win detection, draw handling, and game reset functionality.

## 🚀 Live Demo

🔗 https://anurag-rathod.github.io/Tic-Tac-Toe-Game/

---

## 📌 Features

- 🎯 Two-player turn-based gameplay (X & O)
- 🏆 Automatic win detection
- 🤝 Draw detection
- 🔄 Reset Game functionality
- 🆕 New Game option
- 📱 Responsive design
- ⚡ Built with Vanilla JavaScript (No frameworks)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📂 Project Structure

```text
Tic-Tac-Toe-Game/
│── index.html
│── style.css
│── app.js
```

---

## ▶️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Anurag-Rathod/Tic-Tac-Toe-Game.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
